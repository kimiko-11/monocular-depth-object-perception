### Depth Aware Object Detection

A real-time perception system that combines object detection and monocular depth estimation to estimate the distance of detected objects from a single RGB camera.

This project demonstrates a simplified robotics perception pipeline, where detection results are fused with depth information to generate spatial awareness and a bird’s-eye obstacle map.

The system uses Ultralytics YOLOv8 for object detection and MiDaS for monocular depth estimation, with visualization and processing implemented using OpenCV.

### Features

• Real-time object detection from webcam</br>
• Monocular depth estimation</br>
• Distance estimation for detected objects</br>
• Depth heatmap visualization</br>
• Bird’s-eye obstacle mapping</br>
• Real-time perception pipeline
